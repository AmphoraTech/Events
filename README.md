# Amphora Tech Events

Welcome to **Amphora Tech Events** — materials, formats, and ideas for tech meetups, workshops, and hands-on sessions.

## Contents

- **[Tech events home](workshops/README.md)** — Overview of all Amphora tech events, formats, and how to run them.
- **Workshops** — Detailed workshop plans (theory + hands-on) in the [workshops](workshops/) folder. Each event has its own markdown doc with learning objectives, prep checklists, and facilitation notes.

## Quick links

| What you need | Where to go |
|---------------|-------------|
| See all event types and pick one | [workshops/README.md](workshops/README.md) |
| Run a specific workshop | [workshops/](workshops/) → open the numbered doc (e.g. `01-ctf-on-the-pi.md`) |

## Contributing

Add new event formats or workshops by adding a markdown file under [workshops/](workshops/) and linking it from the [tech events home](workshops/README.md).
